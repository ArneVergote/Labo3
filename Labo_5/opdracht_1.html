Wat zijn de siblings van het < ul > element?

De siblings van < ul > zijn < h1 > en < p >.

Wat zijn de ancestors van het tweede li element?

de ancestors van het tweede < li > element < ul >, < body >, < html >, en het document root element

Wat zijn de descendants van het body element?

< h1 >, < p >, < ul >, en alle drie < li > elementen zijn de descendants.

Hoeveel elementen in een geldig HTML document hebben geen parent?

In een geldig HTML-document heeft elk element een ouder, behalve het root element. Dus er is slechts één element zonder ouder, dat is het root element < html >.

Heeft elk element één of meerdere child elementen?

Elementen kunnen nul of meer child elementen hebben, afhankelijk van de structuur van het HTML-document. Sommige elementen, zoals < img > of < br >, hebben meestal geen child elementen, terwijl andere elementen, zoals < div >, kunnen worden gebruikt om andere elementen te omvatten en kunnen dus meerdere child elementen hebben.
